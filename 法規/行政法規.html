國立成功大學醫技系教學特優教師遴選辦法

  • 940615系務會議通過
  1. 為鼓勵本系教師之教學熱誠，提高教學成效，特定本辦法。
  2. 遴選對象為本系專任教師任教滿3學年者。
  3. 遴選辦法如下:
    1. 每年三月由本系大三、大四學生採無記名投票。
    2. 得票數最高之兩位教師得獲選為優良教師。
  4. 系主任得呈報1-2位優良教師之名單予醫學院，參與院際及校際年度優良教師遴選。
  5. 教師獲「教學傑出」獎後五年內不再予以重覆推薦；獲「教學優良」獎隔年不再重複推薦。
  6. 本辦法於系務會議通過後實施，修正時亦同。

國立成功大學醫學院醫學檢驗生物技術學系教師評量要點

  • 94.02.25系務會議通過
  • 94.06.21第八次校教評會通過
  1. 依據國立成功大學（以下簡稱本校）教師評量要點第四條訂定國立成功大學醫學院醫學檢驗生物 技術學系（以下簡稱本系）教師評量要點（以下簡稱本要點）
  2. 凡本系專任教師，均依本要點接受評量。
    教授及副教授每滿五年接受一次評量，助理教授及講師每滿三年接受一次評量。
    符合下列條件之一者，得免接受評量：
    1. 獲選為中央研究院院士者。
    2. 曾獲頒教育部學術獎或國家講座，本校講座及經本校認可之國內外著名大學講座教授者。
    3. 曾獲國際著名學術獎或在學術上有卓越貢獻，經系教師評審委員會認可者。
    4. 曾獲頒國科會傑出研究獎三次（含）以上，甲（優）等研究獎共十次或研究主持費共十五次（含） 以上者（一次傑出研究獎相當於三次甲等研究獎，一次甲等研究獎助相當於二次研究主持費）。曾 獲選本校教學特優教師者，相當於一次傑出研究獎。
    5. 年滿六十歲（但初聘者除外）。
    6. 曾獲其他教學、研究、服務獎項或其成果具體卓著，經系教師評審委員會認可免接受評量者。
  3. 本系專任教師於接受評量期間，教學、研究、服務符合下列條件之一者，經系教師評審委員會認 可得免接受一次評量：
    1. 於任職期間曾在Impact Factor 5.0以上之期刊或在SCI、SSCI排名百分之五以內的期刊（以發表 當年度該學門之排名計，該領域若少於20種期刊則以該領域排名第一的期刊）發表論文至少三篇 以上，並須為第一作者或通訊作者。
    2. 任職期間經本校或本院推薦出任建教合作醫院之正、副院長或學術研究機構（如中研院、國科會 、國家衛生研究院）及衛生行政機構之一級以上主管且貢獻卓越者。
    3. 本校特聘教授。
    4. 醫學院教學特優教師、醫學院年度最佳論文獎、醫學院新人最佳論文獎、醫學院基礎醫學研究獎、醫學院 服務特優教師，二次勵人獎。
  4. 評量項目及標準：
    1. 教學：
      1. 教學時數：教授每週8小時、副教授及助理教授每週9小時、講師每週10小時。
      2. 教學績效：學生教學反應平均至少3.5，及其他具體事實。
      3. 教師成長活動積點：每年至少6點。
      4. 指導學生、專業知識的應用及其他有關事項。
    2. 研究：
      1. 論文發表情形：
        ①講師及助理教授於三年內需有一篇論文是第一作者或通訊作者。
        ②副教授於五年內需有二篇論文是第一作者或通訊作者。
        ③教授於五年內需要有三篇論文是第一作者或通訊作者。
      2. 專利。
      3. 有無獲得研究計劃並附上個人國科會研究指標。
      3. 　　
    3. 服務：各級行政業務、委員會工作、社團指導、其他學術活動，及臨床 服務。
  5. 必須接受評量之教師應於三月十五日以前提出相關資料由系教師評審委員會進行初審，未提出者，以 該年度未通過評量論。但當年度有留職留薪或留職停薪情形（休假研究、借調、出國講學或進修…）不 在校致未能提出者，俟返校服務後順延辦理。
  6. 新聘教授及副教授依本校教師聘任辦法第五條規定，通過續聘者，視同通過一次評量。 　　　　新聘助理教授及講師依本校教師聘任辦法第五條規定，於規定年限內，不需接受評量。通過升等時，視 同通過一次評量。
  7. 本系專任教師依本要點評量不通過者，於次一年起不予晉薪且不得申請休假研究、借調、在外兼職兼課 ，被評量不通過人員一年後可申請辦理再評量，自再評量通過之次年起恢復晉薪，得申請休假研究、借調 、在外兼職兼課。
  8. 凡最近一次評量不通過者不得提出升等；不得延長服務年限；且不得擔任各級教評會委員，如為現任委員 者，由候補委員遞補之。
  9. 應接受評量年數之計算，不包括留職停薪期間，但借調期間折半計算。通過升等教師，依其升等後職稱， 自該學年度起算其應接受評量年數。對應接受評量年數之計算有疑義時由人事室解釋。
  10. 教師評審委員若為受評當事人，應迴避與自身評量有關之討論及議決。會議之召開均須達三分之二（含） 以上委員出席，始得開議；經出席委員二分之一（含）以上同意始得決議。
  11. 受評量教師對初審結果不服者，得向院教師評審委員會提出書面申復。對申復結果不服者，得向校教師 評審委員會提出書面申復，對申復結果不服者，得向校教師申訴評議委員會提出書面申訴。
  12. 本要點未盡事宜，悉依相關法規辦理。
  13. 本要點經系務會議通過，送院教師評審委員會轉呈校教師評審委員會核備後實施，修正時亦同。

  3. 申請升等資格
    1. 修訂之教育人員任用條例實施前(86.3.21)聘任者，如繼續任職而未中斷，得逕依原升等辦法送審。
    2. 各職級之升等，都需有一定的服務年資，惟具有本校教師聘任辦法中較高職級之條件，其服務年資 得不受本項規定。
    3. 主論文需為三年內之第一作者或通訊作者。
    4. 在國外研究期間完成之論文不得送審升等教授。
    5. 研究或綜合式論文積分需達下列標準，才能提出申請：

      升等職級

      基本分數

      教授

      380

      副教授

      300

      助理教授

      230
    6. 選擇A組教學積分需達85分，選擇C組服務積分需達85分，選擇B組研究積分需達滿分標準 【教授500，副教授400，助理教授300】。
  4. 申請步驟及評審程序
    1. 本系老師於每年六月底前，準備下列資料向系教評會提出申請：
      1. 個人學術生涯說明：以3至5頁A4紙敘述其學術生涯中曾探索與尋求解決的問題，其解 決的策略、方法及歷程及解決的成果，這些成果對人類知識的貢獻，以及未來的學術 規劃與目標。
      2. 教學課目、時數及其它有助評估教學績效之相關資料。
      3. 研究積分算式、論文副本及其SCI、SSCI排名，並附上個人國科會研究指標。
      4. 列舉服務之內容及事項。
      5. 教師資格審查履歷表一式二份
      6. 教師升等表格(一)至表格(七)
      7. 代表著作及參考著作各三份
    2. 升等候選人須在系內作研究專題報告，並向系教評會提出送審論文及論文“研究成果”之說 明，如為教材著作得說明其中之研究成份及其重要性。
    3. 系教評會委員根據送審之資料，就各位候選人之教學、研究、服務作詳細而公平之評審及 評分，各項評分須分別超過70分，且教學＋服務＋研究≧75（分數係以去除各項最高及最 低分數後的平均，再加計其百分比計之），始能獲得投票權。除此之外並參考個人學術生 涯說明、院內研究專題報告及及候選人之品德操守等，在系教評會充分討論後投票。系教 評會委員投票後，決定升等人選及其優先次序，向本院教師評審委員會推薦。候選人須得 系教評會出席委員三分之二(含)以上之同意始能獲得推薦。
    4. 每年各階層之升等名額，依照學校之規定計算。
    5. 申請者雖經系內推薦，但未通過院教師評審委員會之評審者，第二年得重新申請。
    6. 本辦法經系務會議通過，報請院長提請校教師評審委員會核備後實施，修正時亦同。

  1. 本系為辦理系主任推選事宜，依據本校組織規程第二十八條及第二十八條之一訂定本辦法。
  2. 本系應於主任任期屆滿前五個月或即將出缺或已出缺時，組成推選委員會，進行主任推選事宜，此委員會於新主任就任後自動解散。
  3. 推選委員會由七人組成，且副教授以上委員不得少於二分之一，推選委員不能或不便參與推選工作時應於推選工作開始前聲明，並依得票多寡遞補。委員會召集人由委員互選一人擔任。
  4. 推選委員由本系專任教師採無記名方式投票，選出五位委員，其中至少有一位系外委員，並另報請院長聘請二位推選委員。
  5. 推選委員會應主動公開徵求及尋覓主任候選人，並依本辦法召開會議進行推選工作，於主任任期屆滿前至少一個月前推薦人選一至二人呈請院長轉請校長擇聘。只推選一人者，如未獲同意，應重新推選，但以一次為限。
  6. 系主任候選人應具國內外副教授以上資格。
  7. 系主任候選人若為推選委員會委員，則須退出該委員會。
  8. 系主任候選人應提供學經歷、著作目錄、學術成就及對本系發展之意見等資料送請推選委員會參考，委員會得邀請候選人面談，並向全系教師徵求書面意見。
  9. 推選委員會開會時，委員應親自出席，不得委託他人為代表，須有三分之二以上委員出席始得開議。須有出席委員三分之二以上贊成，始得作成推薦候選人選之決議。
  10. 主任任期三年，得連任一次，若擬續任者於第一次任期屆滿前六個月由院長聘請院內教授二人及本系教師互推三人，共五人組成評估委員會。評估委員會決定是否同意主任續任，若全部委員的半數同意續任則連任，若不同意續任則重新辦理主任推選作業，評估委員會應將結果送至院及校核備。
  11. 主任在任期間如發生重大事件致有不適任之虞者，由系務會議代表二分之一以上或全系教師二分之一以上連署提案，經系務會議代表三分之二以上出席，出席代表三分之二以上議決通過，報請校長核定，解除其系主管職務，並依本辦法辦理系主任推選。
  12. 推選委員為無給職，校外委員得酌支車馬費。
  13. 本辦法經系務會議及院務會議通過，報請校長核備後實施，修正時亦同。

國立成功大學醫學檢驗生物技術學系導師制度施行細則

  1. 本細則依「國立成功大學導師制實施辦法」第九條之規定訂之。
  2. 為落實導師輔導工作，提昇教育品質，培養德智兼備之人才，達成大學教育目的，特定訂本細則。
  3. 本系專任講師以上教師均有擔任導師之義務，導師工作為獎勵、升等教師評鑑之參考。系主任為系總導師，負責全系性的導師活動及調諧，但不擔任家族或班級導師。
  4. 導師制度之編組方式為家族導師與班級導師並行制（每一個家族約１３－１５人）依位家族導師、每班級設兩位班級導師，系主任則為系總導師。
  5. 導師經費每學期每導生新台幣1160元，分為導師人事費與學生輔導活動費，分配比例為8：2，導師人事費928元由學務處撥入導師帳戶，學生輔導活動費232元月撥入本系帳戶，專用於學生輔導、教師與學生聯誼活動及系學會活動等。
  6. 家族導師之職責如下：
    1. 了解導生性向、興趣、人格特質、生活與家庭狀況，協助導引其身心發展。必要時可連絡心理師實施性向、興趣或人格測驗。
    2. 協助導生課業學習、選課及生涯規劃等事宜。若導生學期成績不及格達三 分之一以上或受記過處分時，結合學務處各組、家長或有關人員施予適切輔導。
    3. 每學年至少應與每位學生單獨會談一次，並得於每學期安排一次團體聚餐討論。
    4. 於每學期結束前兩週，依據各導生平日生活言行表現，評定其操行成績，送交學務處生活輔導組。
    5. 若導師發現導生出現適應欠佳、偏差行為、或其他特殊事件時，請告知其家長、監護人或緊急聯絡人，並轉介給心理師實施心理諮商與治療。
    6. 協助學生資料之建立：
      1. 學生綜合資料表：每年更新學生資料，於該生畢業後，送交系辦公室歸檔。
      2. 導談記錄表：導師平時訪談時作記錄，每學年結束後（六月底），交由系辦公室打字電腦存檔。
      3. 因實施輔導所獲得導生個人或家庭資料，相關人員依法有保密的義務。
  7. 班級導師之職責如下：
    每個月得舉行一次班級導師時間，協調與討論全班性事宜，並繳交導師時間會議記錄。另運用課餘舉行師生座談、聯誼或其他團體活動，以增進師生感。
  8. 若導生有下列情形之一者，當事人、發現者或事發單位，請迅速通報校安中心（分機55555），協助進行危機處理與後續處置。
    1. 其言行、情緒或精神異常，有可能發生自我傷害或傷害他人之行為時，應依據「本校校園精神疾病及自我傷害個案處置之作業流程」處理。
    2. 其遭受意外、交通事故或其他需緊急就醫之情形，應依據「本校學生急診通報系統」處理。
    3. 其行為有嚴重危害校園安全及安寧，而屬於校園緊急安全事件時，應依據「本校校園事件通報系統暨處理流程」處理。
    4. 系主任導師之職責如下：
    5. 系主任為全系總導師，統籌全系性導師活動及為本系對外之窗口。
    6. 協調授課老師（特別是系外老師）與各導師之連繫，對於學習情形不佳的學生主動告知其家族導師，以即時加以輔導。
    7. 每學期舉辦一次全系的系主任導師時間，與全系的老師學生座談。
    8. 導師會議與每個月的系務會議合併舉行，可提案討論工作實施情形，檢討並改進導師制。
    9. 本細則經系務會議通過後施行，修正時亦同。

成功大學醫學檢驗生物技術學系課程委員會組織章程

  • 800802系務會議訂定
  • 960621系務會議修訂
  • 990520系務會議修訂
  1. 宗旨：
    1. 本系開授課程遵行教育部及校訂定之課程學分標準，及發揚本系特色之系訂課程，乃設立課程委員會推動執行。
    2. 本系課程委員會（以下簡稱本委員會）旨在合理安排課程之學分及學期分佈、聯繫授課教師、教學之規劃與實施，以期達成學生學習之效果與教學目標。
  2. 組織：本會成員由本系專任老師、學生代表大學部及研究所各1人、學者專家或產業界人士1人組成，並聘委員中1人為執行秘書。委員均為無給職，任期二年，連聘得連任。
  3. 職責：
    1. 依據教育部及校部、系訂標準，修訂本系必修及選修課程。
    2. 督導蒐集各年度學期之各科課程進度表、教學內容大綱，存檔以供參考。
    3. 本委員會得設計問卷，於每學期末進行各科教學評鑑，並統計結果，供主任及授課老師參考。
    4. 編訂先修科目，提報系務會議決議之。
    5. 整合各課程之開設及教學內容範圍之協調，以彰顯本系之特色及迎合本系中長程發展方向之所需。
  4. 本章程經系務會議通過後施行，修改亦同。

成功大學醫學檢驗生物技術學系兼任及合聘教師規定

  • 93.11.25系務會議通過
  • 94.10.20系務會議通過
  • 98.04.16系務會議通過
  • 101.11.22系務會議通過
  1. 依據「國立成功大學校內單位間教師合聘辦法」訂定本系合聘教師聘任辦法。
  2. 兼任老師聘任比照本系及本校教師聘任辦法辦理。
  3. 兼任及合聘教師必須符合下列原則：
    1. 認同本系教學原則及目標。
    2. 具有部定助理教授（含）以上資格之教師。
    3. 有研究計劃能提供學生研究所需材料之教師。
  4. 兼任及合聘教師之權利：
    1. 可以指導本系之研究生，每學年最多一名。（需直接指導研究生）
  5. 兼任及合聘教師之義務：
    1. 每學年在本系至少授課4小時。
    2. 參與本系碩士班進度報告活動。
    3. 出席碩士班每學期專題討論1/3以上課程。
    4. 指導本系研究生參與的研究成果若經發表，該論文必須有該研究生與本系之掛名。
  6. 本系每學年合聘教師員額不得超過本系專任教師員額之1/2。
  7. 本系專任老師得依教學需要向本系系務會議推薦合適人選並檢具其履歷、著作，經系務會議同意，合聘雙方教評會審查通過後，簽請學校發聘。
  8. 兼任及合聘教師原則上一年一聘，並得視實際教學及研究需要，經合聘雙方教評會同意後，循本校教師續聘程序辦理續聘。
  9. 本辦法經本系系務會議通過後施行，修正時亦同。

國立成功大學醫學檢驗生物技術學系教師評審委員會設置辦法

  • 800430系務會議訂定
  • 850307系務會議修訂
  • 860603系務會議通過
  • 860729校教評會核備
  • 960621系務會議修訂

  1. 依據國立成功大學各學系（所）教師評審委員會設置辦法及有關法令規定，設置國立成功大學醫學檢驗生物技術學系教師評審委員會(以下簡稱本會)。
  2. 本會設委員本會設委員七人，由下列人員組成之：
    1. 當然委員：系主任(兼召集人)
    2. 推選委員：由系務會議推選本系專任教授六人，已同時擔任院、校級教評會委員者，不得再擔任之。如本系專任教授人數不足六人，得由系務會議自校內外選聘專長相近之教授或本系專任副教授擔任委員，惟教授人數不得少於三分之二。委員任期一年，連選得連任。委員之推選辦法另定之。
  3. 本會以每學期開會一次為原則，必要時得召開臨時會議。
  4. 本會審議左列事項：
    1. 專任教師之聘任、聘期、升等、解聘、停聘、不續聘、 資遣原因之認定等事項。
    2. 兼任教師之聘任、聘期、解聘、停聘等事項。
    3. 名譽教授之聘任事項。
    4. 其他有關教師評審之重要事項，及有關規定需由本會審議之事項。
  5. 教師之聘任、解聘、停聘、不續聘，由本會依本系教師聘任初審辦法審議通過後，提院教師評審委員會審議。教師之升等由本會依本系教師升等初審辦法審查通過後，向院教師評審委員會推薦。其他教師評審重要事宜，由本會依本系發展需要及相關規定審查，通過後依相關規定辦理。
  6. 本會開會時非有應出席委員三分之二(含)以上之出席不得開議，非經出席委員三分之二(含)以上同意不得決議。（有應迴避之情事者，不計入出席委員人數）。
    並得視事實需要，由召集人邀請其他有關人員列席報告或說明。
    副教授委員（含當然委員、召集人），對教授級之聘任、升等審議案，應迴避不得參與表決，其他一般審議事項則不受限制 。
  7. 本會委員在審議案件時，如本人或其配偶、前配偶、四親等內之血親或三親內之姻親為當事人或有利害關係時應自行迴避。未自行迴避者，得經本會決議請該委員迴避。
    本會委員就審議案件如認為有前項以外之特殊事由應迴避時，得向本會申請迴避。
    有具體事實足認本會委員就審議案件有偏頗之虞者，當事人得向本會申請該委員迴避，並應舉其原因事實。
  8. 本辦法未盡事項悉依本校有關辦法辦理。
  9. 本辦法經系務會議通過報請院長核轉校教師評審委員會核備後實施。修正時亦同。
